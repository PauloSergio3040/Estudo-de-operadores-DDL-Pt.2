SQL DDL – Estruturas e Manutenção MySQL

Este repositório contém um script SQL com exemplos de criação e alteração de estruturas no MySQL. Inclui tabelas, chaves primárias e estrangeiras, índices, views, procedures, triggers, tabelas temporárias, além de operações de backup interno, ajustes de AUTO_INCREMENT e manutenção geral de banco.

O objetivo é servir como material de estudo e prática para quem está aprendendo DDL e administração básica de banco de dados.
